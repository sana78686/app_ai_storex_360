# Shipping Carrier Logos

Please place your shipping carrier logo images here with the following filenames:

- `dhl.png` - DHL logo
- `hermes.png` - Hermes logo
- `dpd.png` - DPD logo
- `gls.png` - GLS logo
- `deutsche-post.png` - Deutsche Post logo

**Recommended specifications:**
- Format: PNG with transparency or SVG
- Size: Approximately 60-80px width (height will scale proportionally)
- Background: Transparent or white
- Aspect ratio: Maintain original logo proportions
