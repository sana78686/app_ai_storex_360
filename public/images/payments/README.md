# Payment Method Logos

Please place your payment method logo images here with the following filenames:

- `visa.png` - Visa logo
- `maestro.png` - Maestro logo
- `paypal.png` - PayPal logo
- `klarna.png` - Klarna logo
- `giropay.png` - Giropay logo
- `ueberweisung.png` - Banküberweisung logo
- `vorkasse.png` - Vorkasse logo

**Recommended specifications:**
- Format: PNG with transparency or SVG
- Size: Approximately 60-80px width (height will scale proportionally)
- Background: Transparent or white
- Aspect ratio: Maintain original logo proportions
